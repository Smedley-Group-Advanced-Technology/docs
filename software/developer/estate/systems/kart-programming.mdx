## Overview

Kart Programming was a proof of concept to abstract away the complexities of DVT for kart programming. The project has not been active since 2024 and is considered out of use.

## Description

Kart Programming consisted of an Excel spreadsheet connected to a locally hosted Python API running as a Windows service, which communicated with the kart over CANOpen to standardise the programming process.
